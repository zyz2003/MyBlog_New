  <files>eslint.config.js, .prettierrc, .prettierignore, package.json</files>
  <action>
    1. 安装依赖:
       pnpm add -D eslint-plugin-vue eslint-config-prettier @typescript-eslint/parser typescript-eslint globals

    2. 创建 `eslint.config.js` (ESLint v9 flat config):
       - 导入 globals, typescript-eslint, eslint-plugin-vue, eslint-config-prettier
       - 配置 globals: { ...globals.browser, ...globals.node }
       - 配置 typescript: tseslint.configs.recommended, files: ['**/*.{ts,mts,cts}']
       - 配置 vue: pluginVue.configs['flat/recommended'], files: ['**/*.vue']
       - 配置 prettier: eslintConfigPrettier 最后应用

    3. 创建 `.prettierrc`:
       - semi: false, singleQuote: true, tabWidth: 2, printWidth: 100
       - useTabs: false, bracketSpacing: true, arrowParens: always
       - trailingComma: es5

    4. 创建 `.prettierignore`:
       - node_modules, dist, .nuxt, .output, *.min.*

    5. 更新根 `package.json` scripts:
       - lint: eslint .
       - lint:fix: eslint . --fix
       - format: prettier --write .
       - format:check: prettier --check .

    使用 ESLint v9 flat config 格式，不使用 eslintrc。

  </action>

    <automated>pnpm lint --max-warnings=0 && echo "ESLint config valid"</automated>
  </verify>
  <done>

    - pnpm format:check 通过
  </done>
