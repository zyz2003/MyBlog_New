1. `apps/site/composables/useAdminApi.ts`:
   - Import `useAuthStore` from `~/stores/auth`
   - Import `useAuth` from `~/composables/useAuth`
   - Create a helper function `adminFetch<T>(url: string, options?: RequestInit): Promise<T>`:
     - Reads `token` from auth store
     - Sets `Authorization: Bearer {token}` header
     - Uses `$fetch` from Nuxt (auto-imported)
     - On 401 error: call `useAuth().logout()`, re-throw
     - On other errors: extract error message from response body, throw
   - Export `useAdminApi()` returning:
     - `get: <T>(url: string, params?: Record<string, any>) => adminFetch<T>(url, { method: 'GET', params })`
     - `post: <T>(url: string, body?: any) => adminFetch<T>(url, { method: 'POST', body })`
     - `put: <T>(url: string, body?: any) => adminFetch<T>(url, { method: 'PUT', body })`
     - `del: <T>(url: string) => adminFetch<T>(url, { method: 'DELETE' })`

     - Drafts: `i-heroicons-pencil-square` icon, draft count

   - Each card: white bg, rounded, shadow, icon (colored), label (gray), value (large bold)
