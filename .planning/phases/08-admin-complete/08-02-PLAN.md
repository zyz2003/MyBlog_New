API endpoints:
- GET /api/media?page=1&pageSize=20 -> paginated media list
- POST /api/media/upload -> multipart form upload (field: file)
- DELETE /api/media/:id -> delete media file
- GET /api/themes -> list all themes with active status
- POST /api/themes/:name/activate -> activate a theme
- GET /api/plugins -> list all plugins with enabled status
- POST /api/plugins/:name/enable -> enable plugin (body: { config })
- POST /api/plugins/:name/disable -> disable plugin
- PUT /api/plugins/:name/config -> update plugin config (body: { config })
