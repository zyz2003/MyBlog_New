## Objective

创建后台管理布局系统，包括侧边栏导航、顶部栏、以及分组式导航菜单。

---

## UI/UX Design Flow

**本 Plan 是后台布局核心，必须使用 Pencil + Skills 组合工作流程：**

| 步骤 | 工具 | 用途 |
|------|------|------|
| 1. 整体布局 | `mcp__pencil__batch_design` | 创建侧边栏 + 主内容区线框图 |
| 2. 设计灵感 | `ui-ux-pro-max` | 获取组件设计灵感、交互模式参考 |
| 3. 代码实现 | `frontend-design` | 生成高质量 Vue 3 + Tailwind 组件代码 |
| 4. 主题配色 | `theme-factory` | 定义后台配色方案、CSS 变量 |
| 5. 截图验证 | `mcp__pencil__get_screenshot` | 对比设计稿和实现效果 |
| 6. 代码审查 | `simplify` | 审查组件代码质量、优化复用 |

**设计文件位置：** `docs/design/phase-05/` 目录存放 .pen 设计文件

### Task 1: 创建 AdminLayout 组件

**文件：** `apps/site/components/layouts/AdminLayout.vue`

使用 shadcn-vue Sidebar 组件：
- SidebarProvider - 侧边栏提供者
- Sidebar - 侧边栏容器
- SidebarHeader - 侧边栏头部（Logo）
- SidebarContent - 侧边栏内容
- SidebarFooter - 侧边栏底部（用户信息）
- 主内容区域 overflow-auto

**功能：**
