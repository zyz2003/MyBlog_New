## Tasks

### Task 1: 安装 shadcn-nuxt 模块

```bash
pnpm dlx nuxi@latest module add shadcn-nuxt
```

**验证：**
- `apps/site/nuxt.config.ts` 中包含 `shadcn-nuxt` 模块
- `pnpm install` 成功

### Task 2: 初始化 shadcn-vue

```bash
pnpm dlx shadcn-vue@latest init
```

**配置选项：**
- Style: Default
- Base Color: Slate
- CSS Variables: Yes
- Tailwind Config: auto-detect

**验证：**
- `apps/site/components.json` 创建
- `apps/site/tailwind.config.js` 更新

### Task 3: 安装基础 UI 组件

```bash
pnpm dlx shadcn-vue@latest add button card table input label select dialog dropdown-menu separator badge avatar navigation-menu sidebar form toast
```

**组件列表：**
- `button` - 按钮
- `card` - 卡片容器
- `table` - 数据表格
- `input` - 输入框
- `label` - 标签
- `select` - 下拉选择
- `dialog` - 对话框
- `dropdown-menu` - 下拉菜单
- `separator` - 分隔线
- `badge` - 徽章
- `avatar` - 头像
- `navigation-menu` - 导航菜单
- `sidebar` - 侧边栏
- `form` - 表单
- `toast` - 通知

**验证：**
- `apps/site/components/ui/` 目录包含所有组件
- 组件可正常导入使用

### Task 4: 安装 Pinia 和插件

```bash
pnpm add pinia @pinia/nuxt pinia-plugin-persistedstate
```

**配置：**
- `apps/site/nuxt.config.ts` 添加 Pinia 模块
- 配置 persistedstate 使用 localStorage

**验证：**
- `pnpm -C apps/site test:run` 通过

### Task 5: 安装图标库

```bash
pnpm add lucide-vue-next
```

**验证：**
- 图标可正常导入：`import { Home } from 'lucide-vue-next'`

### Task 6: 安装表单验证库

```bash
pnpm add vee-validate @vee-validate/zod zod
```

**验证：**

### Task 7: 创建测试目录结构

**验证：**
- 目录结构创建完成
- 创建基础的 test setup 文件

### Task 8: 创建 Pinia 配置

**文件：** `apps/site/plugins/pinia.ts`

```typescript
import { createPinia, defineStore } from 'pinia'
import piniaPluginPersistedstate from 'pinia-plugin-persistedstate'

export default defineNuxtPlugin((nuxtApp) => {
  const pinia = createPinia()
  pinia.use(piniaPluginPersistedstate)
  nuxtApp.vueApp.use(pinia)
})
```

**验证：**
- Pinia store 可正常使用
- persistedstate 正常工作
