## UI/UX Design Flow

**本 Plan 包含复杂 UI 组件，必须使用 Pencil + Skills 组合工作流程：**

| 步骤 | 工具 | 用途 |
|------|------|------|
| 1. 整体布局 | `mcp__pencil__batch_design` | 创建文章列表页、编辑页线框图 |
| 2. 设计灵感 | `ui-ux-pro-max` | 获取表格、表单、编辑器设计灵感 |
| 3. 代码实现 | `frontend-design` | 生成高质量 Vue 3 + Tailwind 组件代码 |
| 4. 主题配色 | `theme-factory` | 定义表格样式、表单配色 |
| 5. 截图验证 | `mcp__pencil__get_screenshot` | 对比设计稿和实现效果 |
| 6. 代码审查 | `simplify` | 审查组件代码质量、优化复用 |

**设计文件位置：** `docs/design/phase-05/` 目录存放 .pen 设计文件

## Tasks

### Task 1: 创建文章列表组件

**文件：** `apps/site/components/admin/posts/PostList.vue`

使用组件：
- shadcn-vue Table
- Badge（状态显示）
- Button（操作按钮）

**功能：**
- 表格展示文章列表
- 分页（页码分页，10/20/50/100 可选）
- 筛选（按分类、标签、状态）
- 搜索（按标题）
- 排序（按创建时间）
- 渐进式加载动画

**API 集成：**
- GET /api/v1/posts?limit=&offset=&category=&tag=&status=&search=&sort=&order=

**测试：** `apps/site/tests/components/admin/posts/PostList.test.ts`

### Task 2: 创建文章列表页面

**文件：** `apps/site/pages/admin/posts/index.vue`

**功能：**
- 使用 AdminLayout
- 页面标题 + 新建按钮
- PostList 组件
- 筛选栏

**测试：** `apps/site/tests/pages/admin/posts/index.test.ts`

### Task 3: 创建简单编辑器组件

**文件：** `apps/site/components/admin/posts/SimpleEditor.vue`

**设计决策（来自 CONTEXT.md）：**
- Phase 5: 简单编辑器（textarea + Markdown 工具栏）
- Phase 9: 升级到 TipTap + Vditor

**功能：**
- Markdown 工具栏：
  - H1/H2/H3 标题
  - 加粗/斜体
  - 引用

  - 插入图片
  - 任务列表
- textarea 内容编辑
- 实时预览（可选）

**测试：** `apps/site/tests/components/admin/posts/SimpleEditor.test.ts`

### Task 4: 创建文章表单组件

**文件：** `apps/site/components/admin/posts/PostForm.vue`

**字段：**
- 标题（必填）
- Slug（自动生成，可编辑）
- 分类选择（Select）
- 标签输入（支持多选）
- 封面图上传
- SEO 标题
- SEO 描述
- 内容（SimpleEditor）
- 状态（草稿/发布）

**验证：**
- 标题：必填，最少 2 字符
- Slug：唯一性检查
- 分类：必填

**测试：** `apps/site/tests/components/admin/posts/PostForm.test.ts`

### Task 5: 创建文章编辑页面

**文件：** `apps/site/pages/admin/posts/[id].vue`

**功能：**
- 新建模式（id = new）
- 编辑模式（id = 文章 ID）
- 表单验证
- 自动保存草稿（每 30 秒）
- 保存提示 Toast
- 离开确认（有未保存更改）

**API 集成：**
- GET /api/v1/posts/:id (编辑模式)
- POST /api/v1/posts (新建)
- PUT /api/v1/posts/:id (更新)

**测试：** `apps/site/tests/pages/admin/posts/[id].test.ts`

### Task 6: 创建批量操作组件

**文件：** `apps/site/components/admin/posts/BulkActions.vue`

**功能：**
- 多选 checkbox
- 批量删除
- 批量发布/下架
- 全选/取消全选

**API 集成：**
- DELETE /api/v1/posts/bulk (批量删除)
- PUT /api/v1/posts/bulk (批量更新状态)

**测试：** `apps/site/tests/components/admin/posts/BulkActions.test.ts`

### Task 7: 创建分类标签选择器

**文件：** `apps/site/components/admin/posts/CategoryTagSelector.vue`

**功能：**
- 分类下拉选择（支持树形）
- 标签多选（支持创建新标签）
- 已选项目显示

**API 集成：**
- GET /api/v1/categories (获取分类列表)
- GET /api/v1/tags (获取标签列表)
- POST /api/v1/tags (创建新标签)

**测试：** `apps/site/tests/components/admin/posts/CategoryTagSelector.test.ts`

### Task 8: 创建封面图上传组件

**文件：** `apps/site/components/admin/posts/CoverImageUploader.vue`

**功能：**
- 图片预览
- 从媒体库选择
- 直接上传
- 移除图片

**API 集成：**
- GET /api/v1/media (媒体库)
- POST /api/v1/media (上传)

**测试：** `apps/site/tests/components/admin/posts/CoverImageUploader.test.ts`
